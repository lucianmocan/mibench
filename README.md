MiBench
=======

Original code with some build / makefile / configuration changes to use: -O3 and compile with LLVM 21.1.8 (clang) (using symlinks so that gcc calls actually are clang calls, makes it easy to use gcc instead).

This is the original MiBench code as downloaded from http://www.eecs.umich.edu/mibench/index.html.

The version modified for embedded targets and running automtically is in the
embedded branch
